# Site institucional — Fluxus Tecnologia

Site institucional da Fluxus Tecnologia (`fluxustecnologia.com.br`).

- **Stack:** Next.js 16 (App Router) · React 19 · TypeScript · Tailwind CSS v4 · pnpm
- **Arquitetura e decisões técnicas:** ver [docs/ARQUITETURA.md](docs/ARQUITETURA.md)
- **Deploy:** Railway Pro (container, `output: 'standalone'`)

## Desenvolvimento

```bash
pnpm install
pnpm dev      # http://localhost:3000
pnpm build    # build de produção
pnpm lint
```
